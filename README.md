# Michael Berthold Portfolio Starter

GitHub Pages compatible Jekyll starter with:
- English + German homepage
- shared menu and footer
- publications subpages
- responsive layout close to the supplied design direction

## Upload to GitHub Pages

1. Create a new repository on GitHub.
2. Upload all project files to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and the **/(root)** folder.
6. Save.
7. Wait 1–2 minutes.
8. Your site will appear on the GitHub Pages URL shown there.

## Edit content

- Homepage EN: `en/index.md`
- Homepage DE: `de/index.md`
- Publications EN: `en/publications.md`
- Publications DE: `de/publications.md`
- Menu labels: `_data/navigation.yml`
- Shared header: `_includes/header.html`
- Shared footer: `_includes/footer.html`
- Styles: `assets/css/styles.css`

## Replace images

Replace the SVG files in `assets/img/` with final exported images from the design:
- `portrait.svg`
- `speaker.svg`
- `talks.svg`
- `book-1.svg`
- `book-2.svg`
- `book-3.svg`
